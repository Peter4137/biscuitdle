## Image Requirements

- Square images work best (the viewer will crop to square)
- Minimum recommended size: 400x400px
- JPG or PNG format

## Where to Find Images

1. **Take your own photos** - Most authentic!
2. **Wikimedia Commons** - Search for the biscuit name
3. **Wikipedia** - Each biscuit page usually has an image

Once you add images here, update `src/data/biscuits.js` to use local paths:

```javascript
image: '/biscuits/digestive.jpg',
```
